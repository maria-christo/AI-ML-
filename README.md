# AI Student Performance Predictor

An AI-powered web application that predicts a student's exam score based on study habits and generates personalized study recommendations using Google's Gemini AI.

## Features

- Predicts exam scores using a trained Machine Learning model
- Takes student inputs such as:
  - Hours Studied
  - Attendance Percentage
  - Previous Scores
  - Sleep Hours
- Generates AI-powered study recommendations using Gemini
- Provides:
  - Study Plan
  - Daily Schedule
  - Revision Strategy
  - Exam Tips
  - Areas for Improvement
- Interactive web interface built with Streamlit

## Tech Stack

- Python
- Scikit-learn
- Pandas
- Streamlit
- Google Gemini API
- Joblib

## Project Workflow

1. Collect and preprocess student performance data
2. Train a Machine Learning regression model
3. Save the trained model using Joblib
4. Build a Streamlit interface for user interaction
5. Generate personalized academic guidance using Gemini AI
